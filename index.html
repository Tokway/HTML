HTML elements

Завдання виконав Юрій Кравченко

<html></html>

Вказує програму перегляду сторінок, що це HTML документ.
<head></head> Визначає місце, де міститься різна інформація, що не відображається в тілі документа.
Тут розміщується тег назви документа та теги для пошукових машин.
<body></body> Визначає видиму частину документа
<title></title> Поміщає назву документа у зміст програми перегляду сторінок
<pre></pre>

Обрамляє попередньо відформатований текст.
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6> 		Шість заголовків різного рівня,
від самого великого до маленького по порядку зростання
<b></b> Створює жирний текст
<i></i> Створює похилий текст
<q></q> відображає в тексті цитати, браузер бере текст у скобки
<strong></strong> також відображає текст жирним шрифтом
<tt></tt> Створює текст - стиль друкарської машинки
<cite></cite> Використовується для цитат, як правило, похилий текст.
<font size=?></font> Встановлює розмір тексту від 1 до 7.
<font color=?></font> Встановлює колір тексту за допомогою значення кольору.⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀
<p></p> Створює новий параграф
<sub></sub> Відображає шрифт у нижньому індексі поля
<sup></sup> Відображає шрифт у верхньому індексі поля
<br> Вставляє переклад рядка.
<blockquote></blockquote> Створює відступи з обох сторін тексту.
<a href="URL"></a> Створює посилання на інші документи або частину поточного документа. Тут URL адреса посилання, текст - текст посилання.
<a name="NAME"></a> Відзначає частину тексту як місце переходу за гіперпосиланням у документі.
<a href="#NAME"></a> Створює гіперпосилання на позначену частину документа.
<ol></ol> Створює нумерований перелік
<ul></ul> Створює ненумерований перелік
<div></div> Тег, який використовується для форматування великих блоків тексту HTML документа. Як правило використовується у таблицях стилів
<img src="ім'я">

Додає зображення у HTML документ
<hr> Додає HTML документ горизонтальну лінію.
<table></table> Створює таблицю.
<button></button> Cтворює на веб-сторінці кнопку.
<tr></tr> Визначає рядок у таблиці.
<td></td> Визначає окремий осередок у таблиці.
<th></th> Визначає заголовок таблиці (нормальний осередок з відцентрованим жирним текстом)
<table border=#>

Визначає товщину рамки таблиці.
<table cellpacing=#> Задає відстань між осередками таблиці.
<table cellpadding=#> Задає відстань між вмістом комірки та її рамкою.
<table width=#> Встановлює ширину таблиці у пікселах чи відсотках від ширини документа.
<tr align=?> або <td align=?> Встановлює вирівнювання осередків таблиці, приймає значення: left, center, чи right.
<tr valign=?> або <td valign=?> Встановлює вертикальне вирівнювання для осередків таблиці, приймає значення: top, middle, або bottom.
<td colspan=#> Вказує кількість стовпців яке об'єднане в одному осередку. (за замовчуванням = 1)
<td rowspan=#> Вказує кількість рядків, що об'єднано в одному осередку. (за замовчуванням = 1)
<form></form> Створює форми.
<select multiple name="NAME" size=?></select> Створює меню, що скролується. Size встановлює кількість пунктів меню, яке буде показано на екрані, інші будуть доступні при використанні прокручування.
<select name="NAME"></select> Створює спадне меню
<textarea name="NAME" cols=40 rows=8></textarea> Створює вікно для введення тексту. Columns вказує ширину вікна; rows вказує його висоту.
<input type="checkbox" name="NAME"> Створює checkbox. За тегом слідує текст.
<input type="radio" name="NAME" value="x"> Створює кнопку radio. За тегом слідує текст.
<input type=text name="foo" size=20> Створює рядок для введення тексту. Параметром Size вказується довжина символів.
<input type="submit" value="NAME"> Створює кнопку "Прийняти"
<input type="reset"> Створює кнопку "Скасувати"
